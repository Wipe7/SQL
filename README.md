# Projetos de Estoque - EBAC

Este repositório contém projetos desenvolvidos durante o curso de SQL da EBAC, utilizando os bancos de dados **PostgreSQL** e **SQLite**. Os projetos têm como objetivo praticar conceitos de modelagem, manipulação de dados, criação de índices e análise de desempenho de consultas.

## Estrutura do Repositório

- **estoque-postgres/**  
  Scripts e exemplos utilizando PostgreSQL, incluindo análise de índices e performance.

- **estoque-sqlite/**  
  Scripts e exemplos utilizando SQLite para gerenciamento de estoque.

## Principais Funcionalidades

- Criação de tabelas para controle de estoque, clientes e vendas.
- Inserção, atualização e exclusão de dados.
- Consultas SQL para análise de dados.
- Criação e análise de índices para otimização de consultas (PostgreSQL).
- Scripts de teste de performance com `EXPLAIN ANALYZE`.
